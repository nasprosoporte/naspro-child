# NASPRO Child Theme

Este repositorio contiene el child theme de WordPress para la web de NASPRO.

## Objetivo

Construir una web profesional para NASPRO en WordPress, sin Elementor y sin WooCommerce.

WordPress se usará para:
- estructura visual
- contenido
- SEO
- landing principal
- experiencia de marca

Shopify se usará para:
- productos
- packs
- suscripciones
- checkout
- pagos
- pedidos
- gestión de clientes

## Contexto actual

De momento solo hay un producto principal:

- NASPRO 30 tiras nasales

Más adelante se añadirán más productos a la web.

La idea es que el cliente pueda comprar en pocos clics:
- una unidad
- un pack
- una suscripción mensual

## Reglas importantes

- No usar Elementor.
- No usar WooCommerce.
- No inventar precios exactos.
- No inventar tiempos de envío.
- No inventar claims médicos.
- Evitar que el producto parezca un producto sanitario.
- Mantener un enfoque wellness, deporte, descanso o uso diario.
- Si falta un dato, dejarlo como pendiente.
- Trabajar paso a paso.
